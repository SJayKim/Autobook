# 0. 들어가며

이 책은 AI 코딩 에이전트인 Claude Code와 OpenAI Codex CLI를 실무에서 안전하게 운용하기 위한 설정·확장·운영 레시피를 모은 쿡북입니다. 두 도구의 동작 원리를 이론적으로 풀어 내기보다, 같은 문제를 두 도구에서 어떻게 다르게 해결하는지를 짧고 구체적인 단원으로 짚어 갑니다.

Phase 1과 2에서는 두 도구의 기본 구조 차이와 프로젝트 컨텍스트 설계를 다룹니다. Phase 3부터 5까지는 권한·샌드박스, hook, MCP 같은 통제와 확장 지점을 정리합니다. Phase 6과 7은 작업 분해와 피드백 루프, Phase 8은 팀 도입과 보안·트러블슈팅을 다룹니다.

각 단원은 가능한 범위에서 settings.json·config.toml·CLAUDE.md·AGENTS.md 같은 실제 파일 예시를 곁들이고, 같은 의도를 두 도구로 각각 구현하는 짝맞춤 형태를 유지합니다. 책에서 '하네스 엔지니어링'이라는 용어가 등장하면, 모델 자체가 아니라 모델을 둘러싼 외부 설정·통제·도구·피드백 구조를 가리킵니다.

다음 단원에서는 먼저 두 도구를 한눈에 비교하면서 어떤 작업에 어느 쪽이 잘 맞는지부터 정리하겠습니다.
