# 3.2 샌드박스와 격리

이 섹션의 토픽과 학습 목표는 다음과 같습니다.

- **3.2.1 Codex CLI sandbox 모드 활용** — read-only, workspace-write, danger-full-access 세 sandbox 모드를 작업별로 선택할 수 있다
- **3.2.2 devcontainer로 격리 실행** — devcontainer나 Docker로 두 도구를 격리 실행하는 구성을 작성할 수 있다
- **3.2.3 위험 명령 차단 레시피** — rm -rf, git push --force, curl | sh 같은 위험 명령을 두 도구에서 동시에 차단하는 설정을 작성할 수 있다
