# 4.2 MCP: 모델과 도구의 연결

이 섹션의 토픽과 학습 목표는 다음과 같습니다.

- **4.2.1 MCP의 구조: 호스트·클라이언트·서버** — MCP의 호스트-클라이언트-서버 구조와 각 역할을 설명할 수 있다
- **4.2.2 MCP의 세 가지 기본형: 도구·리소스·프롬프트** — MCP 서버가 노출하는 도구(tools)·리소스(resources)·프롬프트(prompts)를 구분해 설명할 수 있다
- **4.2.3 MCP 전송 계층: stdio와 HTTP** — MCP의 전송 방식(stdio, 스트리밍 HTTP)과 선택 기준을 설명할 수 있다
- **4.2.4 MCP 서버 만들기** — 간단한 MCP 서버가 도구를 정의하고 노출하는 과정을 단계별로 설명할 수 있다
- **4.2.5 기존 도구를 MCP 서버로 감싸기** — 이미 있는 API나 함수를 MCP 서버로 감싸 표준 인터페이스로 노출하는 방법을 설명할 수 있다
- **4.2.6 MCP 보안과 신뢰** — MCP 서버 연결에서 발생하는 보안 위험과 권한·신뢰 경계를 설명할 수 있다
