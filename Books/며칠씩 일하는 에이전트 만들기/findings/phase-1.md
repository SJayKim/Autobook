# Phase 1 그라운딩 브리프 — 에이전트와 2026년의 전환

집필 에이전트는 이 브리프의 사실관계·용어·프레이밍을 일관되게 따른다. 각 토픽은 curriculum.json의 learning_content를 모두 다룬다.

## 책 전체의 프레이밍 (모든 토픽이 공유)
- 책의 목표: "며칠씩 일하는 에이전트"(장기 실행 에이전트, long-running agent). 초기 에이전트는 몇 분짜리 단발 작업을 했지만, 이제는 며칠씩 작동하며 전체 시스템을 구축하고 인간은 핵심 결정 지점에서만 전략적 감독을 제공하는 방향으로 진화한다.
- 그 목표를 떠받치는 네 인프라 축: (1) 컨텍스트 엔지니어링 (2) 레이어드 메모리 (3) 프로토콜 표준(MCP·A2A) (4) 하네스 엔지니어링. 책은 1~4축을 차례로 쌓아 Phase 5에서 장기 실행 에이전트로 종합한다.
- 일관 용어: '에이전트', '모델', '하네스', '컨텍스트 윈도우', '토큰', '루프', '도구(tool)'. 영어 약어는 처음 등장 시 한글 설명 병기.

## 핵심 정의
- AI 에이전트: 모델이 한 번의 응답에 그치지 않고 여러 번의 행동(도구 호출)을 이어 목표를 달성하려는 구성. 네 구성요소 = 모델(두뇌) + 도구(행동 수단) + 메모리(기억) + 루프(반복 제어).
- LLM의 본질: 문자열을 입력받아 문자열을 내보내는 함수에 가깝다. 스스로 파일을 열거나 네트워크 요청을 보내지 못한다. 바깥과의 연결은 모두 하네스가 만든다.
- 모델 대 하네스: 모델 = 가중치·추론 능력(학습으로 정해짐). 하네스 = 모델 바깥의 실행 환경·도구·통제·피드백. 같은 모델이라도 하네스에 따라 성능·안전성이 달라진다. 비유: 엔진(모델)과 차체·운전 보조(하네스).
- 도구 사용(tool use): 모델이 정해진 형식(도구 이름 + 인자)으로 외부 기능 호출을 요청하면 하네스가 실행하고 그 결과를 문자열로 컨텍스트에 돌려준다.

## 에이전트 루프와 ReAct
- 에이전트 루프: 입력 → 추론 → 행동(도구 호출) → 관찰(결과 환류) → 다시 추론 … 종료 조건까지 반복. 한 바퀴를 '턴(turn)'이라 부른다. 종료 조건이 없거나 부실하면 무한 루프·비용 폭주 위험.
- ReAct: Reasoning(추론) + Acting(행동)을 한 사이클에 결합. Thought → Action → Observation 반복. 추론만 하면 환각(없는 사실을 지어냄)을 못 잡고, 행동만 하면 맹목적이다. 관찰(실제 결과)을 다시 읽어 추론을 교정하는 것이 ReAct의 핵심. 책 전체에서 ReAct는 에이전트 루프의 기본 패턴으로 둔다.
- 루프가 길어질 때의 문제(장기 실행의 근본 난점): ① 컨텍스트 누적·팽창 ② 오류 전파·누적 ③ 호출 비용 증가 ④ 목표 표류(처음 목표에서 점점 벗어남). 이 네 가지가 각각 컨텍스트/하네스/FinOps/메모리로 이어진다.

## 2026년의 네 가지 전환 (Phase 1.3, 각 축의 예고)
1. 프롬프트 → 컨텍스트 엔지니어링: 프롬프트 한 줄을 다듬는 일에서, 컨텍스트 윈도우 '전체'를 유한 자원으로 관리하는 일로 무게중심 이동. 컨텍스트 엔지니어링은 프롬프트 엔지니어링을 대체한다기보다 그것을 포함하는 상위 규율이다.
2. 세션 → 영속 레이어드 메모리: 대화 한 번(세션)이 끝나면 사라지는 기억에서, 세션을 넘어 지식을 영속 저장하는 계층형 메모리로.
3. 프로토콜 표준화(MCP·A2A): 모델-도구 연결(MCP), 에이전트-에이전트 통신(A2A)의 표준화. 새로운 아키텍처(멀티에이전트) + 새로운 표준 + 새로운 경제학(에이전트용 FinOps) + 새로운 거버넌스의 결합.
4. 장기 실행 + 하네스 엔지니어링: 며칠씩 도는 에이전트를 안정적으로 굴리기 위한 self-correction·compaction·rollback·human approval boundary를 갖춘 오케스트레이션 인프라. 그 자체가 별도 연구 분야로 부상.

## 집필 주의
- Phase 1은 도입부다. 뒤 Phase의 내용을 미리 깊게 설명하지 말고 '예고' 수준으로만 짚는다(각 축은 해당 Phase에서 상세히 다룸).
- 전문가 전제 금지. 모든 새 용어는 배경부터 풀어 설명.
