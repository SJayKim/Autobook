# Agentic AI 보안 설계: 권한을 가진 신뢰할 수 없는 실행 주체를 다루는 법

이 책은 도구를 부르고 메모리를 쌓으며 며칠씩 스스로 일하는 LLM 에이전트를 실제 시스템으로 올릴 때 필요한 보안을 다룹니다.
