# 2 LLM 보안의 기초

LLM 애플리케이션 일반의 위협 지형을 OWASP LLM Top 10으로 정리하고, 프롬프트 인젝션을 직접 재현·설명하며, 과도한 에이전시가 왜 에이전트 보안의 출발점인지 연결할 수 있다.

이 Phase는 다음 섹션으로 구성됩니다.

- 2.1 LLM 애플리케이션의 위협 지형
  - 2.1.1 OWASP LLM Top 10 개관
  - 2.1.2 프롬프트 인젝션(LLM01): 직접 인젝션
  - 2.1.3 민감정보 유출과 부적절한 출력 처리
  - 2.1.4 공급망과 데이터·모델 오염
  - 2.1.5 벡터·임베딩 취약점
- 2.2 에이전시로 넘어가는 다리
  - 2.2.1 과도한 에이전시(LLM06): 에이전트 보안의 출발점
  - 2.2.2 왜 모델 계층 방어만으로는 부족한가
