# 2.1 LLM 애플리케이션의 위협 지형

이 섹션의 토픽과 학습 목표는 다음과 같습니다.

- **2.1.1 OWASP LLM Top 10 개관** — OWASP Top 10 for LLM Applications가 무엇을 목록화한 표준인지 설명하고, 열 개 항목의 큰 갈래를 짚을 수 있다
- **2.1.2 프롬프트 인젝션(LLM01): 직접 인젝션** — 직접 프롬프트 인젝션이 어떻게 시스템 지시를 덮어쓰는지 메커니즘을 설명하고 간단한 예로 재현 원리를 서술할 수 있다
- **2.1.3 민감정보 유출과 부적절한 출력 처리** — 민감정보 유출(LLM02)과 부적절한 출력 처리(LLM05)가 무엇이며 서로 어떻게 이어지는지 구별해 설명할 수 있다
- **2.1.4 공급망과 데이터·모델 오염** — LLM 공급망 취약점(LLM03)과 데이터·모델 오염(LLM04)이 어디서 발생하는지 설명할 수 있다
- **2.1.5 벡터·임베딩 취약점** — 벡터·임베딩 취약점(LLM08)이 RAG 구조에서 어떻게 위험이 되는지 설명할 수 있다
