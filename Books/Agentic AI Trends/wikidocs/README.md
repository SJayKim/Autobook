# Agentic AI: 자율 에이전트의 구조와 최신 동향

불과 몇 년 전만 해도 우리는 AI에게 "질문"을 했습니다.
하지만 이제 우리는 AI에게 "임무"를 맡깁니다.

이 책은 Agentic AI의 내부 작동 원리를 체계적으로 풀어냅니다.
에이전트 아키텍처, 메모리, 도구 사용, 계획 수립, 자기 교정, 그래프 RAG까지 6개 파트, 67개 토픽으로 구성되어 있습니다.

## 대상 독자

- LLM 기반 에이전트의 작동 원리가 궁금한 개발자
- Agentic AI 시스템을 설계하려는 엔지니어
- AI 에이전트 기술의 최신 동향을 파악하고 싶은 분
