# Phase 6. 응용 시스템과 운영 자동화

이 Phase에서는 RAG, 에이전트 등 복합 LLM 시스템을 구축하고 전체 ML/LLM 운영 체계를 자동화하는 방법을 다룹니다. 이 단계를 마치면 실무에서 마주하는 다양한 LLM 응용 시나리오를 설계하고, 운영 전반을 자동화할 수 있습니다.

**6.1 RAG 파이프라인**에서는 RAG 아키텍처, RAG 평가 메트릭, 환각 감지, 컨텍스트 엔지니어링을 다룹니다. 외부 지식을 검색하여 LLM 응답의 정확성을 높이는 파이프라인을 설계하고 평가하는 방법을 익힙니다.

**6.2 LLM 안전과 비용 운영**에서는 LLM 가드레일, LLM 게이트웨이와 모델 라우팅, LLM 비용 최적화 전략을 다룹니다. LLM 시스템의 안전성을 확보하면서 비용을 효율적으로 관리하는 실무 전략을 살펴봅니다.

**6.3 파이프라인 자동화**에서는 파이프라인 오케스트레이션 도구, 자동 재훈련 시스템 설계, 인프라 코드화를 다룹니다. 모델의 훈련, 평가, 배포를 자동으로 반복하는 파이프라인을 구축하고 인프라를 코드로 관리하는 방법을 익힙니다.

**6.4 에이전틱 AI**에서는 에이전틱 AI 개념과 패턴, LangGraph를 다룹니다. LLM이 도구를 호출하고 계획을 수립하며 스스로 판단하는 에이전트 시스템의 구조와 구현 방법을 살펴봅니다.
