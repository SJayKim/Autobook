# 들어가며

머신러닝 모델을 연구 환경에서 만드는 것과 실제 서비스에서 운영하는 것은 전혀 다른 문제입니다. 학습 스크립트를 한 번 실행하는 데 그치지 않고, 데이터가 바뀔 때마다 모델을 재훈련하고, 새 버전을 안전하게 배포하며, 성능 저하를 실시간으로 감지해야 합니다. 여기에 대규모 언어 모델(LLM)이 더해지면 프롬프트 관리, 토큰 비용 통제, 환각 방지 같은 새로운 과제가 추가됩니다.

이 책은 그러한 운영 과제를 체계적으로 다루기 위해 쓰였습니다. MLOps의 기본 원리에서 출발하여 Apache Kafka 기반의 실시간 데이터 파이프라인을 설계하고, LLM을 포함한 모델의 서빙과 모니터링을 거쳐, RAG와 에이전틱 AI까지 이어지는 전체 운영 체계를 한 권에 담았습니다.

이 책의 대상 독자는 머신러닝이나 딥러닝의 기초 개념은 알고 있지만, 이를 프로덕션에 적용하는 과정이 낯선 분들입니다. 데이터 엔지니어, ML 엔지니어, 백엔드 개발자 가운데 실시간 데이터 처리와 모델 운영에 관심이 있는 분이라면 이 책에서 실질적인 도움을 얻을 수 있습니다.

책은 총 6개의 Phase로 구성되어 있습니다. Phase 1에서는 MLOps, LLMOps, Kafka의 기초 개념을 익힙니다. Phase 2에서는 Kafka 심화 기능과 스트림 처리, 피처 엔지니어링을 다룹니다. Phase 3에서는 실험 관리, 파인튜닝, 모델 경량화를 배웁니다. Phase 4에서는 다양한 배포 전략과 LLM 추론 최적화를 살펴봅니다. Phase 5에서는 드리프트 감지와 LLM 관측 가능성을 다루고, 마지막 Phase 6에서는 RAG, 가드레일, 파이프라인 자동화, 에이전틱 AI를 통해 전체 운영 체계를 완성합니다.

각 단원은 앞선 단원의 내용을 전제로 하므로, 처음부터 순서대로 읽는 것을 권장합니다. 그럼 Phase 1의 첫 번째 주제인 'MLOps의 정의와 필요성'부터 시작하겠습니다.
