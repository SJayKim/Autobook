# 5.2 LLM 관측 가능성

이 섹션에서는 LLM 시스템에 특화된 모니터링과 관측 기법을 다룹니다. 다음 세 개의 토픽으로 구성되어 있습니다.

**5.2.1 LLM 관측 가능성 체계**에서는 LLM 프로덕션 시스템에서 프롬프트, 응답, 토큰, 지연시간을 추적하는 관측 가능성 체계를 살펴봅니다. OpenTelemetry, OpenLIT, Prometheus, Grafana 등을 활용한 트레이스 수집 방법을 이해할 수 있습니다.

**5.2.2 LLM 비용 추적과 최적화**에서는 토큰 사용량, 모델별 단가, 요청 건수 기반으로 LLM 추론 비용을 추적하고 관리하는 방법을 살펴봅니다. 비용 대시보드, 예산 알림, 비용 귀속 등의 실무 기법을 이해할 수 있습니다.

**5.2.3 LLM 드리프트 탐지**에서는 임베딩 분포 변화와 응답 품질 저하를 시계열로 감지하는 LLM 드리프트 탐지 방법을 살펴봅니다. 임베딩 벡터 분포 변화와 통계적 검정 적용을 통한 자동 경보 체계를 이해할 수 있습니다.
