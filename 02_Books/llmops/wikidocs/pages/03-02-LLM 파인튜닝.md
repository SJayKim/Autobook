# 3.2 LLM 파인튜닝

이 섹션에서는 사전학습된 대규모 언어 모델을 도메인에 맞게 조정하는 다양한 기법을 다룹니다. 다음 네 개의 토픽으로 구성되어 있습니다.

**3.2.1 LLM 파인튜닝 기초**에서는 사전학습 모델을 도메인 특화 데이터로 파인튜닝하는 주요 방식과 선택 기준을 살펴봅니다. 전체 파인튜닝, PEFT, SFT 등의 개념과 도메인 적응, 데이터 준비, 훈련 비용을 이해할 수 있습니다.

**3.2.2 LoRA와 QLoRA**에서는 LoRA의 저랭크 행렬 학습 원리와 QLoRA의 4비트 양자화 결합 메커니즘을 살펴봅니다. 동결 가중치에 어댑터를 추가하는 방식으로 VRAM을 절감하는 원리를 이해할 수 있습니다.

**3.2.3 RLHF**에서는 RLHF의 보상 모델 학습과 PPO 기반 정책 최적화 과정을 단계별로 살펴봅니다. 인간 선호 데이터 수집부터 KL 발산 페널티까지 이어지는 정렬 과정을 이해할 수 있습니다.

**3.2.4 DPO**에서는 DPO가 보상 모델 없이 선호 쌍으로 직접 정책을 최적화하는 원리와 RLHF 대비 장점을 살펴봅니다. Bradley-Terry 모델 기반의 단순화된 정렬 방식을 이해할 수 있습니다.
