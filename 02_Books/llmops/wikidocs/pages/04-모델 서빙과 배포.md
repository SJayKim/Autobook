# Phase 4. 모델 서빙과 배포

이 Phase에서는 ML/LLM 모델을 다양한 전략으로 프로덕션에 배포하고 효율적으로 서빙하는 방법을 다룹니다. 이 단계를 마치면 서비스 중단 없이 모델을 교체하고, 대규모 추론 요청을 효율적으로 처리할 수 있습니다.

**4.1 배포 전략**에서는 블루-그린 배포, 카나리 배포, 섀도 배포와 A/B 테스트, Kubernetes 기반 ML 배포를 다룹니다. 모델을 안전하게 프로덕션에 올리고, 문제가 발생했을 때 빠르게 롤백하는 다양한 전략을 비교합니다.

**4.2 LLM 추론 서빙**에서는 LLM 추론 최적화 기법, vLLM, 배치 추론 패턴, 실시간 모델 추론 패턴을 다룹니다. 대규모 언어 모델의 추론 비용과 지연시간을 줄이는 핵심 기술과 아키텍처 패턴을 익힙니다.
