# Phase 3. 모델 개발과 실험 관리

이 Phase에서는 ML/LLM 모델을 체계적으로 훈련하고 최적화하며 실험과 버전을 관리하는 방법을 다룹니다. 이 단계를 마치면 재현 가능한 실험 환경을 구축하고 모델을 효율적으로 개선할 수 있습니다.

**3.1 실험 관리와 버전 관리**에서는 실험 추적 도구, 모델 레지스트리, 데이터 버전 관리, 프롬프트 버전 관리를 다룹니다. 모델 훈련 과정에서 발생하는 하이퍼파라미터, 메트릭, 아티팩트를 체계적으로 기록하고 관리하는 방법을 익힙니다.

**3.2 LLM 파인튜닝**에서는 LLM 파인튜닝 기초, LoRA와 QLoRA, RLHF, DPO를 다룹니다. 사전학습된 대규모 언어 모델을 도메인에 맞게 조정하는 다양한 기법과 그 원리를 이해합니다.

**3.3 모델 경량화**에서는 양자화 기법, 가지치기와 지식 증류, 온라인 학습과 지속적 모델 갱신을 다룹니다. 모델의 크기와 연산량을 줄이면서도 성능을 유지하는 기법과, 스트리밍 데이터로 모델을 점진적으로 갱신하는 방법을 살펴봅니다.
