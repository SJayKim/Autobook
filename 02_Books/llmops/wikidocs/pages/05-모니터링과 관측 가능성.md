# Phase 5. 모니터링과 관측 가능성

이 Phase에서는 프로덕션 ML/LLM 시스템의 성능, 드리프트, 비용을 체계적으로 모니터링하고 이상을 감지하는 방법을 다룹니다. 이 단계를 마치면 모델이 프로덕션에서 기대한 대로 동작하는지 지속적으로 확인하고, 문제를 조기에 발견할 수 있습니다.

**5.1 ML 모니터링**에서는 ML 모니터링의 계층 구조, 데이터 드리프트 감지, 챔피언-챌린저 패턴을 다룹니다. 인프라부터 비즈니스 지표까지 이어지는 모니터링 계층을 이해하고, 데이터 분포가 변할 때 이를 감지하여 모델을 안전하게 교체하는 방법을 익힙니다.

**5.2 LLM 관측 가능성**에서는 LLM 관측 가능성 체계, LLM 비용 추적과 최적화, LLM 드리프트 탐지를 다룹니다. LLM 시스템에 특화된 프롬프트/응답 추적, 토큰 비용 관리, 임베딩 분포 변화 감지 등의 기법을 살펴봅니다.
