# LLMOps, Kafka, MLOps: 실시간 데이터 파이프라인에서 LLM 운영까지

이 책은 머신러닝 모델을 프로덕션 환경에서 안정적으로 운영하는 데 필요한 기술 전반을 다룹니다. MLOps의 기본 원리부터 시작하여 Apache Kafka 기반의 실시간 데이터 파이프라인, 대규모 언어 모델(LLM)의 서빙과 모니터링, 그리고 RAG와 에이전틱 AI까지 이어지는 실무 중심의 내용을 담고 있습니다.

**Phase 1. 기초 개념**에서는 MLOps, LLMOps, Apache Kafka의 핵심 용어와 기본 원리를 다룹니다. 이 단계를 마치면 세 영역의 기본 개념을 자기 말로 설명할 수 있습니다.

**Phase 2. 데이터 파이프라인과 스트리밍**에서는 Kafka 심화 기능, 스트림 처리 엔진, 피처 엔지니어링을 다룹니다. 이 단계를 마치면 Kafka 기반 실시간 데이터 파이프라인을 설계하고 피처 엔지니어링 체계를 구축할 수 있습니다.

**Phase 3. 모델 개발과 실험 관리**에서는 실험 추적, 모델 레지스트리, LLM 파인튜닝, 모델 경량화를 다룹니다. 이 단계를 마치면 ML/LLM 모델을 체계적으로 훈련하고 최적화하며 실험과 버전을 관리할 수 있습니다.

**Phase 4. 모델 서빙과 배포**에서는 블루-그린, 카나리 등 배포 전략과 LLM 추론 최적화를 다룹니다. 이 단계를 마치면 ML/LLM 모델을 다양한 전략으로 프로덕션에 배포하고 효율적으로 서빙할 수 있습니다.

**Phase 5. 모니터링과 관측 가능성**에서는 ML 모니터링, 드리프트 감지, LLM 관측 가능성을 다룹니다. 이 단계를 마치면 프로덕션 ML/LLM 시스템의 성능, 드리프트, 비용을 체계적으로 모니터링하고 이상을 감지할 수 있습니다.

**Phase 6. 응용 시스템과 운영 자동화**에서는 RAG 파이프라인, LLM 안전과 비용 운영, 파이프라인 자동화, 에이전틱 AI를 다룹니다. 이 단계를 마치면 RAG, 에이전트 등 복합 LLM 시스템을 구축하고 전체 ML/LLM 운영 체계를 자동화할 수 있습니다.
