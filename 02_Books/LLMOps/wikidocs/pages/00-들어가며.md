# 들어가며

대규모 언어 모델(LLM)은 텍스트 생성, 질의응답, 코드 작성 등 다양한 분야에서 눈에 띄는 성과를 보여 주고 있습니다. 하지만 모델을 실험실에서 꺼내 실제 서비스에 올리는 순간, 전혀 다른 차원의 문제가 시작됩니다. 프롬프트를 누가, 어떻게 관리할 것인지, 추론 비용을 어떻게 통제할 것인지, 모델이 거짓 정보를 생성하면 어떻게 대응할 것인지, 개인정보가 모델을 통해 유출되지는 않는지 등 운영 단계에서만 드러나는 과제가 줄줄이 이어집니다.

LLMOps는 바로 이 지점을 다루는 분야입니다. 기존 MLOps가 전통적 머신러닝 모델의 학습-배포-모니터링 파이프라인을 체계화했다면, LLMOps는 LLM 특유의 프롬프트 관리, 벡터 데이터 운영, 모델 서빙 최적화, 평가와 안전성 확보, 비용 관리, 거버넌스까지를 아우릅니다.

이 책은 LLMOps의 전체 영역을 9개 Phase로 나누어 차근차근 설명합니다. Phase 1에서 LLMOps의 정의와 LLM 기본 원리를 다진 뒤, 프롬프트 엔지니어링, 벡터 데이터와 임베딩, RAG, 오케스트레이션과 에이전트, 모델 서빙과 추론 최적화, 파인튜닝, 평가와 품질 보증, 그리고 운영·배포·거버넌스까지 순서대로 진행합니다.

각 단원은 "왜 이것이 필요한가"라는 배경부터 시작하여, 핵심 매커니즘을 단계별로 풀어 설명합니다. 특정 도구나 프레임워크를 다루는 단원에서는 아키텍처의 원리를 먼저 이해한 뒤 실제 설정과 사용법으로 이어지도록 구성했습니다. 이 책을 끝까지 따라가면, LLM 애플리케이션을 프로덕션 환경에서 안정적으로 운영하는 데 필요한 지식 체계를 갖추게 됩니다.
