# Agentic AI 기획자와 엔지니어

AI 에이전트 시대, 기획자와 엔지니어가 알아야 할 모든 것을 담은 실무 교재입니다.

Agentic AI의 기초 개념부터 에이전트 설계 패턴, AI 제품 기획 방법론, 산업별 적용 사례, 거버넌스와 커리어 전략까지 체계적으로 다룹니다.

**대상 독자:** AI 제품을 기획하거나 개발하려는 PM, 기획자, 엔지니어

**구성:**
- Phase 1: 기초 -- Agentic AI와 기획자의 역할 이해
- Phase 2: 핵심 역량 -- 기획자의 실무 도구
- Phase 3: Agentic 기획 실무 -- 에이전트 시스템 설계
- Phase 4: AI 기획 방법론 -- 제품 기획부터 출시까지
- Phase 5: 응용 -- 산업별 적용과 거버넌스
