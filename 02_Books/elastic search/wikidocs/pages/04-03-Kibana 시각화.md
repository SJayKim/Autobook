# 4.3 Kibana 시각화

이 섹션에서는 Kibana를 활용하여 Elasticsearch 데이터를 시각화하는 방법을 다룹니다. Lens로 차트를 생성하고, Dashboard로 종합 뷰를 구성하며, KQL로 데이터를 필터링하는 방법을 익힙니다.

## 토픽 목록

**4.3.1 Kibana Lens**
- Kibana Lens로 드래그앤드롭 방식으로 시각화를 생성할 수 있다

Kibana Lens UI 구성, 데이터 뷰(Data View) 생성, X축/Y축 필드 매핑, 차트 타입 선택(막대, 선, 파이, 히트맵), 시간 필터 연동, Breakdown 레이어, Lens 수식(Formula) 기능을 다룹니다.

**4.3.2 Kibana Dashboard**
- 여러 시각화 패널을 조합한 대시보드를 구성하고 공유할 수 있다

Dashboard 생성 및 패널 추가, 패널 크기 조정 및 배치, 전역 시간 필터와 패널 시간 오버라이드, Controls 패널(드롭다운 필터), 대시보드 URL 공유, PDF/PNG 내보내기, Reporting 스케줄링을 다룹니다.

**4.3.3 KQL (Kibana Query Language)**
- KQL로 Kibana 검색 바에서 필터 조건을 작성할 수 있다
- KQL과 Lucene 쿼리 구문의 차이를 설명할 수 있다

KQL 기본 문법(field: value), AND/OR/NOT 연산자, 와일드카드 사용, 범위 쿼리(field >= value), 중첩 필드 쿼리, KQL vs Lucene 쿼리 구문 비교, Kibana 필터 패널과 KQL 연동을 다룹니다.
