# 8.1 전자상거래 검색

이 섹션에서는 Elasticsearch로 전자상거래 상품 검색 서비스를 구축하는 방법을 다룹니다. 상품 인덱스 매핑 설계부터 자동완성, 오타교정, 동의어 검색, 비즈니스 로직 반영 스코어링까지 실전 수준의 검색 서비스를 구현합니다.

## 토픽 목록

**8.1.1 상품 인덱스 설계**
- 전자상거래 상품 검색을 위한 인덱스 매핑과 분석기를 설계할 수 있다

상품 인덱스 필드 설계(이름, 설명, 카테고리, 가격, 재고), text+keyword 멀티필드 패턴 적용, 카테고리 필드 keyword 타입, 가격/재고 숫자 타입, nested 타입으로 상품 옵션 표현, dense_vector로 이미지 임베딩 저장, 한국어 상품명 분석기 설계, 동의어 사전(동의어 매핑)을 다룹니다.

**8.1.2 자동완성, 오타교정, 동의어 검색**
- 자동완성, 오타교정, 동의어 확장을 통합한 검색 서비스를 구현할 수 있다

search_as_you_type 기반 자동완성, fuzziness: AUTO 오타교정, prefix 쿼리 기반 자동완성 비교, 동의어 필터로 유사어 확장, 스팸 필터링과 불용어 처리, 검색 결과 function_score 부스팅(판매량, 평점), 전체 검색 UX 플로우 설계를 다룹니다.
