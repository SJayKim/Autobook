# 들어가며

Elasticsearch는 오늘날 가장 널리 사용되는 분산 검색·분석 엔진입니다. 전자상거래 플랫폼의 상품 검색, 대규모 로그 분석, 실시간 모니터링 대시보드, 그리고 최근에는 AI 기반 시맨틱 검색까지, Elasticsearch는 다양한 현장에서 핵심 인프라로 자리 잡고 있습니다.

이 책은 Elasticsearch를 처음 배우는 분부터 프로덕션 환경을 직접 운영하는 엔지니어까지 모두를 위해 쓰였습니다. 단순히 API 사용법을 나열하는 데 그치지 않고, 각 기능이 왜 그렇게 동작하는지, 내부 원리는 무엇인지를 함께 설명합니다. 원리를 이해하면 처음 보는 상황에서도 스스로 판단하고 문제를 해결할 수 있기 때문입니다.

책은 8개의 Phase로 구성되어 있습니다. Phase 1에서는 Elasticsearch의 기본 개념과 내부 구조를 다룹니다. Lucene 역색인, 세그먼트, NRT 검색의 원리를 이해하고 나면, 이후 모든 내용이 훨씬 자연스럽게 연결됩니다. Phase 2와 3에서는 데이터를 어떻게 저장하고 검색할지, 매핑 설계와 Query DSL을 상세히 다룹니다. Phase 4에서는 집계와 시각화를, Phase 5에서는 분산 아키텍처를 다루며, Phase 6에서는 실제 운영에 필요한 ILM, 보안, 모니터링을 다룹니다. Phase 7은 벡터 검색과 AI 검색이라는 최신 주제를 다루며, Phase 8에서는 앞서 배운 모든 내용을 실전 시나리오에 적용합니다.

각 토픽은 선수 지식을 명확히 하고 있으므로, 순서대로 읽으면 지식이 자연스럽게 쌓입니다. 이미 기초가 있는 분이라면 필요한 Phase부터 시작해도 좋습니다.

이 책을 통해 Elasticsearch의 동작 원리를 깊이 이해하고, 실무에서 자신감을 갖고 활용할 수 있게 되기를 바랍니다.
