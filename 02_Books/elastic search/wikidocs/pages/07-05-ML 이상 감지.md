# 7.5 ML 이상 감지

이 섹션에서는 Elasticsearch의 머신러닝 기능을 활용한 이상 감지(Anomaly Detection)를 다룹니다. 시계열 데이터에서 자동으로 이상 패턴을 찾아내고, Elastic Security와 연동하여 보안 위협을 탐지하는 방법을 익힙니다.

## 토픽 목록

**7.5.1 이상 감지 Job 설정**
- Kibana에서 이상 감지 Job을 생성하고 결과를 해석할 수 있다

ML Anomaly Detection 개념, 단일 메트릭 Job vs 다중 메트릭 Job, detector 함수(mean, sum, count, rare 등), bucket_span 설정, influencer 필드, anomaly score 해석(0~100), 결과 인덱스(.ml-anomalies-*), Job 스케줄러와 실시간 분석을 다룹니다.

**7.5.2 보안 모니터링 활용**
- 이상 감지 결과를 Kibana SIEM과 연동하여 보안 이상을 탐지할 수 있다

Elastic Security와 ML 통합, SIEM 탐지 규칙과 이상 감지 연동, Entity Analytics 개요, 이상 징후 알림 설정(Watcher 또는 Alerting), 위협 헌팅(threat hunting) 시나리오, 로그인 이상 탐지 예시를 다룹니다.
