# Elasticsearch 완벽 가이드

이 책은 Elasticsearch를 처음 접하는 개발자부터 프로덕션 환경을 운영하는 엔지니어까지, 실무에서 필요한 모든 지식을 체계적으로 다룹니다. 역색인 원리부터 AI 검색까지, 이론과 실습을 함께 익힐 수 있도록 구성하였습니다.

## 책의 구성

이 책은 8개의 Phase로 이루어져 있으며, 각 Phase는 독립된 역량 단위로 설계되었습니다.

**Phase 1 — 기초 (Foundation)**
Elasticsearch를 로컬에 설치하고, 문서를 인덱싱·검색하며, 역색인과 세그먼트 구조를 설명할 수 있게 됩니다.

**Phase 2 — 데이터 모델링 (Data Modeling)**
인덱스 매핑을 설계하고, 텍스트 분석기를 커스터마이징하며, 한국어 형태소 분석을 적용할 수 있게 됩니다.

**Phase 3 — 검색 (Search)**
Query DSL로 복합 검색을 작성하고, 자동완성·하이라이팅·페이지네이션을 구현하며, 관련성 스코어를 튜닝할 수 있게 됩니다.

**Phase 4 — 집계 및 분석 (Aggregations)**
Bucket/Metric/Pipeline 집계를 중첩하여 복합 분석 쿼리를 작성하고, Kibana Dashboard로 시각화할 수 있게 됩니다.

**Phase 5 — 분산 아키텍처 (Distributed Architecture)**
클러스터 노드 역할을 설계하고, 샤드 전략을 수립하며, Cross-Cluster Search 환경을 구성할 수 있게 됩니다.

**Phase 6 — 운영 및 관리 (Operations)**
ILM 정책을 설계하고, 데이터 수집 파이프라인을 구성하며, 보안·스냅샷·모니터링을 포함한 프로덕션 클러스터를 운영할 수 있게 됩니다.

**Phase 7 — AI 검색 (Modern AI Search)**
벡터 검색, ELSER 시맨틱 검색, BM25+kNN 하이브리드 검색을 구현하고 RAG 파이프라인을 통합할 수 있게 됩니다.

**Phase 8 — 실전 활용 (Use Cases)**
전자상거래 검색, ELK 로그 분석, Observability 시스템을 Elasticsearch로 구축할 수 있게 됩니다.
