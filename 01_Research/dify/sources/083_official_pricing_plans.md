# Dify: Plans and Pricing

Source: Official Dify pricing page (dify.ai/pricing)

## Cloud Service Plans

### Sandbox (Free)
- 200 message credits
- 1 Team Workspace
- 1 Team Member
- 5 Apps
- 50 Knowledge Documents
- 50MB Knowledge Data Storage
- 10 Knowledge Request/min
- Standard Document Processing
- 3,000 Trigger Events
- Up to 2 Triggers/workflow
- Standard Workflow Execution
- 10 Annotation Quota Limits
- 30 Days Log History
- 5,000 API Rate Limit/month
- Support OpenAI/Anthropic/Llama2/Azure OpenAI/Hugging Face/Replicate

### Professional ($59/month)
- 5,000 message credits/month
- 1 Team Workspace
- 3 Team Members
- 50 Apps
- 500 Knowledge Documents
- 5GB Knowledge Data Storage
- 100 Knowledge Request Rate Limit/min
- Priority Document Processing
- 20,000 Trigger Events/month
- Unlimited Triggers/workflow
- Faster Workflow Execution
- 2,000 Annotation Quota Limits
- Unlimited Log History
- No Dify API Rate Limit
- Save $118 with annual billing

### Team ($159/month)
- 10,000 message credits/month
- 1 Team Workspace
- 50 Team Members
- 200 Apps
- 1,000 Knowledge Documents
- 20GB Knowledge Data Storage
- 1,000 Knowledge Request Rate Limit/min
- Top Priority Document Processing
- Unlimited Trigger Events
- Unlimited Triggers/workflow
- Priority Workflow Execution
- 5,000 Annotation Quota Limits
- Unlimited Log History
- No Dify API Rate Limit
- Save $318 with annual billing

### Enterprise (Custom pricing)
- Contact sales

## Feature Comparison Across Plans

### Model Provider
All plans: LLM API Load Balancing, Local Models, Model Vendor Management, Covering All Models

### App Studio
All plans: Chatbot, Text Generator, Agent, Chatflow, Workflow, Webapp Icon Customization, App Version Control, Explore Page Templates

### Dify Marketplace
Free for Now (Coming Soon for paid features)

### Knowledge Base (RAG)
All plans: Batch Document Upload, Knowledge Base Sources, Customize Knowledge Pipeline Templates, Add Chunks, Standard / External Knowledge API, Up to 50 Files/Websites per Upload

### App Deploy
All plans: Publish as Webapp, Publish as API

### App Monitor and Feedback
All plans: App Runtime Data Analysis, Standard + Advanced (Langsmith/Langfuse) monitoring
- Sandbox: 30 Days log history
- Professional/Team: Unlimited log history

### Workspace Management
Webapp Branding Customization, Role Management available

### Support
- Sandbox: Community, Help docs
- Professional: Priority Email
- Team: Priority Email
- All plans: SOC Type II Report

## Self-hosted Option
Available via open-source community edition (free) or Enterprise edition (licensed). Dify is also free for students and educators.

## Notes
- Annual billing saves 17%
- All subscription prices exclude applicable taxes
- Education program available
