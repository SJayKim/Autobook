# Dify.AI Showcases AI Innovation at NVIDIA GTC 2025

Published: March 13, 2025

Dify.AI attended NVIDIA's GTC 2025 in San Jose on March 17-21 at Booth #3226, presenting latest advancements: Plugin System, Workflow Mechanism, and NVIDIA NIM Integration.

## What Attendees Could Experience

- **Live Demos**: Watch Dify in action creating AI Agents
- **Use Cases**: Practical demos of successful AI Agent use cases across industries
- **1v1 Expert Talk**: Meet AI specialists to solve implementation challenges
- **Industry Insight**: Industrial insights and enterprise AI solutions in latest trends

## Special Presentation

March 19 (Wednesday) 17:30-19:00 at GTC AWS Startup Kiosk. Dify.AI's Head of DevRel, Chenhe Gu, presented on efficiently building enterprise AI applications with real-world success stories.

## About Dify

Dify.AI is an open-source LLM app development platform. Core concept: define AI applications through prompts, context, and plugins. Features include prompt orchestration, RAG, Workflow, App publishing, and plugin hosting.

### v1.0.0 Release Highlights

- **Plugins Mechanism**: New plugin-based architecture with Models and Tools migrated to Plugins, introducing Agent Strategies, Extensions, and Bundles
- **Workflow Enhancements**: Agent node added for intelligent orchestration
- **Open Ecosystem**: Dify Marketplace launch for thriving plugin ecosystem
