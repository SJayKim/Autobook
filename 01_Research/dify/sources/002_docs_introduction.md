# Introduction - Dify Docs

Dify is an open-source platform for building agentic workflows. It lets you define processes visually, connect your existing tools and data sources, and deploy AI applications that solve real problems.

The name Dify comes from **D**o **I**t **F**or **Y**ou.

## Getting Started Resources

The official documentation is organized into the following entry points:

- **Quick Start**: Start shipping powerful apps in minutes. A step-by-step tutorial walks through creating a multi-platform content generator from scratch, demonstrating powerful Dify nodes to orchestrate sophisticated AI applications.

- **Concepts**: Core Dify building blocks explained, including app types (Workflow, Chatflow, Chatbot, Agent, Text Generator), variables, and DSL.

- **Self Host**: Deploy Dify on your own laptop or server using Docker Compose.

- **Forum**: Trade notes with the community at forum.dify.ai.

- **Changelog**: Track what has changed over past releases via GitHub releases.

- **Tutorials**: Example Dify use case walkthroughs, such as building a customer service bot.

## Platform Positioning

Dify positions itself as an agentic workflow builder that combines Backend-as-a-Service and LLMOps to streamline the development of generative AI solutions, making it accessible to both developers and non-technical innovators. The community has grown to over 180,000 developers and supports 59,000+ end users.
