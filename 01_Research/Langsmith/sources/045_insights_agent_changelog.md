# LangSmith — Discover Patterns in Production Traces with the Insights Agent (GA Announcement)

## 출시 개요

LangSmith Insights Agent가 2025년 10월 23일 일반 공개(GA)되었습니다. 프로덕션 추적 데이터에서 패턴을 자동으로 발견하는 AI 기반 분석 솔루션입니다.

## 세 가지 핵심 분석 모드

**1. 사용 패턴 분석**

일반적인 사용 패턴을 기반으로 클러스터링하여 사용자가 에이전트를 실제로 어떻게 활용하는지 파악합니다.

**2. 실패 모드 감지**

에이전트 오류와 부정적 상호작용의 근본 원인을 자동으로 그룹화합니다.

**3. 커스터마이징**

분석 카테고리, 시간 범위 필터링, 키워드 검색, 새 속성 정의가 가능합니다.

## 대상 고객

LangSmith Plus 및 Enterprise 클라우드 사용자를 대상으로 합니다.

## 시작 방법

먼저 LangSmith에 트레이스 데이터를 기록해야 합니다. 이후 공식 문서와 영상 튜토리얼을 통해 Insights Agent 설정 방법을 확인합니다.
